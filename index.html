My name is Andrew Stephens and I am a Graphic designer studying at Oregon State University. These are projects that have been assigned to me in the past. For each project there is an explanation further discussing the parameters and requirments.

For this project in Typography 1 we were challenged with creating a booklet that follows and displays the swiss style while sharing information and classification of 3 different typefaces from 3 different type families.

For this assignment in production 1 we were tasked with creating a piece that included atleast 2 of earths elements. For this I decided to include all 4 to improve the design and overall message of the piece.

In this assignment we were tasked with coding a website that showcases information on a movement in postmodern Graphic design. I chose to build a website for the NEW WAVE movement, that explains history, a gallery and sources.
